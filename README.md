# Orynvae

**Orynvae** - локальное AI-приложение для людей с идеей истории. Оно помогает задать стартовый синопсис, собрать память истории и запустить интерактивный режим рассказчика, где пользователь влияет на персонажей и события, а AI превращает прожитые сцены в художественный текст.

Приложение работает на машине пользователя и подключается к выбранному AI-провайдеру: LMStudio, Ollama, OpenAI, OpenRouter или любому OpenAI-compatible endpoint. Основной фокус - приватная работа с локальными моделями, без обязательного облачного backend.

## Ключевая идея

Orynvae должен начинаться не с пустой формы и не с жесткого плана произведения. Пользователь описывает базовую идею, AI извлекает стартовую память истории, предлагает мягкие линии развития и запускает режим рассказчика. Финал не обязан быть известен заранее: история раскрывается интерактивно, глава за главой.

## Основные возможности MVP

- Start Story: создание проекта из свободного синопсиса.
- Model Providers: настройки LMStudio, Ollama, OpenAI, OpenRouter и Custom OpenAI-compatible.
- Story Memory: персонажи, места, предметы, правила мира, тайны и канон.
- Story Lines: мягкие линии истории вместо жесткого плана.
- Narrator Mode: интерактивная глава, где AI ведет сцену ходами.
- Session Log: сохранение решений, реплик, событий и последствий.
- Draft Assembly: сборка художественного черновика из сессии.
- Chapter Review: подтверждение фактов, обновление памяти и линий.
- Forecast: возможные направления следующих глав без фиксации финала.
- Read-only без AI: если AI недоступен, приложение позволяет только читать сохраненные проекты.

## Локальный запуск

- Backend: `http://localhost:9001`
- Frontend: `http://localhost:9002`
- Frontend проксирует API-запросы с `/api` на backend.

### Разработка

Требования для первого этапа:

- Python 3.11+
- `uv`
- Node.js
- `pnpm`

Если Codex/PowerShell-сессия не видит глобальные инструменты, подключите локальный bootstrap:

```powershell
Set-ExecutionPolicy -Scope Process Bypass
. .\scripts\tool-env.ps1
uv --version
pnpm --version
```

`scripts\dev.ps1` делает это автоматически. Если локальная политика PowerShell блокирует `.ps1`, запускайте Windows-обертку:

```powershell
.\scripts\dev.cmd
```

Первичная установка:

```powershell
cd backend
uv sync --extra dev
uv run db-init

cd ..\frontend
pnpm install
```

Запуск backend и frontend одной командой:

```powershell
.\scripts\dev.ps1
```

Отдельные команды:

```powershell
cd backend
uv run dev

cd ..\frontend
pnpm dev
```

## Режим рассказчика

Центральный режим Orynvae v2 - интерактивная сессия главы. Пользователь может управлять одним персонажем, несколькими персонажами или вмешиваться как автор. AI ведет сцену, реагирует на решения, сохраняет последствия, а затем помогает собрать лог сессии в художественный текст.
